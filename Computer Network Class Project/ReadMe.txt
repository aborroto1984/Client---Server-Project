To use this project, you first open a server and then open up to 4 clients that automatically connect to the already open server. 

Commands:
-----------------

$help ------  It lists the commands and what they do.
$exit -------  It exits the chat.
$getlist ----  It shows who is currently in the chat.
$getlog ----  It asks the server for a copy of the ChatLog.txt it creates, and it saves it as ChatTranscript.txt
